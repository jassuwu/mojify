# Use Mx437 IBM BIOS as the default export font

Mojify MP4 export will default to a bundled `Mx437_IBM_BIOS` font so exported videos have a deliberate terminal-art identity rather than a generic programming-font look. Because the font comes from the Ultimate Oldschool PC Font Pack under CC BY-SA 4.0, Mojify must track it as a third-party asset with explicit attribution and license documentation.
